---

## ❓ What it can do (right now)

- Elementary school level (easy & medium)
- Operations: addition, subtraction, multiplication, division, exponentiation
- Division accepts fractions (like `1/2`) or decimals
- Choose a specific operation or let it pick randomly
(ik its elementary school😔 ✌️, but thats all i can do for now, trust the process!1!1!1🗣️)
---

## 🤔 How to run it

1. Make sure you have Python 3.6+
2. Install the only dependency:
   ```bash
   pip install openai
   ```
3. Run the program:
   ```bash
   python user.py
   ```
